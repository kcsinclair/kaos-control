# Dashboard Recent Ideas and Defects Widget

Add a new dashboard widget called "Recent Ideas and Defects" that displays the most recent 6 combined ideas and defects, each item clickable to open the corresponding artifact in the artifact view. This widget should be positioned above the existing Recent Activity widget in its column.

The dashboard column layout should be restructured so that the column containing this new widget is wider. The top row of that column should divide into thirds: Stages Pie Chart occupying one third, Status Distribution occupying one third, and the new Recent Ideas and Defects widget occupying one third. The Completion Velocity widget should span two thirds of the width beneath the two pie charts.

This change improves at-a-glance project health visibility by surfacing the latest defects and ideas directly on the dashboard, reducing the need to navigate to individual artifact lists to check recent activity.
