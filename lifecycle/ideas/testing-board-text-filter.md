# Testing Board Text Search Filter

The Testing Board currently lacks the "Filter by text" search capability that is available on other screens in the application. This inconsistency means users cannot quickly narrow down test artifacts by keyword, forcing them to scroll through the full list manually.

The text search filter from other screens should be applied to the Testing Board, allowing users to filter displayed test entries by typing a search term. The filter should behave consistently with its implementation elsewhere — matching against relevant fields such as title, slug, or status.

This is a usability enhancement that brings the Testing Board into parity with the rest of the application and reduces friction when working with large numbers of test artifacts.
