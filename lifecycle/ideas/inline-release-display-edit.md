# Inline Release Display and Editing on Artefact View

Reorder the metadata fields on the artefact view so that the displayed order is: status, priority, and then release. This brings the release field into the natural reading flow alongside the other key artefact properties.

Display the release value inline on the artefact view, and make it editable with a single click. Clicking the release field should present a dropdown list populated with available releases, allowing the user to assign or change the release directly from the artefact view without navigating away.

This reduces friction when triaging or planning artefacts, keeping release assignment fast and contextual alongside status and priority edits.
