## Raw Idea

disable side menu hover over display

## Idea

The side menu currently shows additional content or expands when the user hovers over it. This hover-triggered behaviour can be disruptive or unintended, particularly when the user is navigating past the menu to reach other content.

The idea is to disable or remove the hover-over display so the side menu only responds to explicit interactions such as clicks or keyboard navigation, giving the user more deliberate control over when the menu expands or reveals additional items.
