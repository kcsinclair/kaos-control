# Stages Distribution Pie Chart

Add a Stages Distribution pie chart widget to the dashboard, positioned beside the existing Status Distribution widget on the left side of the layout. Each slice represents one lifecycle stage (e.g. ideas, requirements, backend-plans, frontend-plans, test-plans, tests, releases) and shows the count of artifacts at that stage.

Pie slices must be clickable, filtering the artifact list or navigating to a view scoped to the selected stage — consistent with the interactive behaviour of the existing Status Distribution widget.

The chart should use the same visual language (colours, card styling) as surrounding widgets to maintain a cohesive dashboard aesthetic.
