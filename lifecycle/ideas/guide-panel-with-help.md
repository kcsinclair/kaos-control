# Guide Panel on Right with help for current screen

## Raw Idea

A guidance panel on the right side with contextual help for the screen you are in, can be enabled/disabled with a guidance switch on the header/menu bar.

The panel will show a summary of what is needed as well as options for what you are doing, e.g. when you are filling in the frontmatter, it is shows the values so you can see.

## Idea

A collapsible guide panel on the right side of the UI that provides contextual help based on the screen the user is currently viewing. The panel can be toggled on or off via a switch in the header or menu bar, allowing users to keep it visible while learning and hide it once familiar.

The panel adapts its content to the current context — for example, when editing artifact frontmatter, it displays the valid vocabulary for each field (statuses, types, lineage conventions) so the user can reference allowed values without leaving the editor. For other screens such as the graph view or kanban board, it would summarise the available actions and relevant workflow rules.

This feature supports both onboarding new users who need guidance on the lifecycle model and experienced users who occasionally need a quick reference, without disrupting the main workspace layout.
