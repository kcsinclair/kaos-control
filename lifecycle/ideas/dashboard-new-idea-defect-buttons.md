# Add New Idea and New Defect Buttons to Dashboard Top Menu

The dashboard's top menu should include "New Idea" and "New Defect" quick-action buttons so users can capture ideas or log defects without first navigating to the artifacts page. This reduces friction for the most common entry-point actions in the lifecycle workflow.

On the artifacts page, the order of the existing buttons should be adjusted so that "New Idea" appears first (left) and "New Defect" appears second (rightmost). This ordering reflects the relative frequency and logical priority of the two actions — ideas are the primary input to the lifecycle, defects are secondary.
