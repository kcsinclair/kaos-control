# Roadmap Backlog Panel and Unscheduled Column Placement

When viewing the Roadmap, the Unscheduled release should appear as the final (rightmost) column with no date, and its corresponding Gantt bar should anchor to the bottom-right corner of the Gantt chart, making it visually distinct from time-boxed releases.

The panel currently shown beneath the Gantt chart for Unscheduled items should be repurposed as a Backlog panel. This panel displays all undefined (unassigned) artifacts, one card per item, and is labelled "Backlog" rather than "Unscheduled".

Clicking any card in the Backlog panel opens the existing artifact edit modal, allowing users to quickly assign backlog items to a release without navigating away from the Roadmap view.
