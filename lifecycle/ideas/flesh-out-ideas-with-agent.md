# flesh out ideas with agent

When triggered the agent reviews and idea and can flesh it out with you, the result would be a better defined idea, NOT a detailed requirement.

The original idea could stay in the top of the markdown file, the agent would/could add missing frontmatter and then a new section with the additional details, to get the idea ready for requirements analysis.

The process here is high level idea -> fleshed out idea -> clarifying questions -> detailed requirements.

So the fleshing out process would be more interactive, should not require a lot of time.
