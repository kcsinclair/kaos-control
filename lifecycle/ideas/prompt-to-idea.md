# prompt to idea

You talk to an LLM and prompt in your vagueness and the LLM redrafts your ramblings into a better defined idea, asks some simple questions, not too detailed, if needed to make sure the idea is clear.

The agent comes up with the slug and creates the idea file with all the meta data automatically.

Essentially removing friction from ideation.
