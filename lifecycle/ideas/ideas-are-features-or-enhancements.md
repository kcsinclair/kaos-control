# ideas are features or enhancements

As an idea is developed it should be given a feature name, but not all ideas will be features, some ideas will be enhancements.

During the requirements analysis phase the idea should be identified as a new feature or an enhancement to an existing feature.

Features should be listed, documented and have defects raised against them.

Features would be part of the graph as well, all artefacts related to a specific feature.
