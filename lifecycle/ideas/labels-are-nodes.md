# Labels are Nodes

When displaying the 2D and 3D graph, labels should be nodes with a unique colour and any artifact using that label would create an edge.

There should be a control in the GUI to disable label nodes.

When displaying the modal for a node, the labels should be displayed as well as the priority.

It should be possible to change the priority directly.

Priority should be displayed on artifact node as a ring around the node, with high being red, medium being orange, normal being green and low blue. 

The colours of the nodes also needs to be updated to match the different types of nodes.  Currently this is not matching the defined types and many nodes are grey.
