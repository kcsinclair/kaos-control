# Front Page Projects Not Listed in Alphabetical Order

## Reproduction Steps

1. Open the kaos-control web UI.
2. Observe the list of projects displayed on the front/home page.

## Expected Behaviour

Projects should be displayed in ascending alphabetical order by project name.

## Actual Behaviour

Projects are not sorted alphabetically; they appear in an unordered or insertion-order sequence.
