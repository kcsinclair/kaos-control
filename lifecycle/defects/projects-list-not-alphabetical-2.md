# Projects List Not Sorted Alphabetically

## Reproduction Steps

1. Launch the application and navigate to the Projects page.
2. Observe the order in which projects are listed.

## Expected Behaviour

Projects should be displayed in ascending alphabetical order by project name, making it easy to locate a specific project at a glance.

## Actual Behaviour

Projects are not sorted alphabetically; they appear in an unordered or arbitrary sequence (e.g. insertion or registration order).
